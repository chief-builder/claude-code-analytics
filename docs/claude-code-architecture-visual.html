Key Insights from mitmproxy Analysis

  • Tool execution happens LOCAL (not on Anthropic servers)
  • MCP tools (browser automation) also run locally via stdio
  • Each tool_use has unique ID matching its tool_result
  • Opus can request multiple tools in parallel
  • 99%+ cache hit rate after first turn (system prompt cached)
  • Context grows: 27K → 42K tokens over session
